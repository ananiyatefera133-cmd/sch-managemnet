# School Fee Management System (Simple Sample)

This is a basic beginner-friendly PHP + MySQL project for XAMPP.

## Folder Structure

- `config/` - Database configuration
- `database/` - SQL file
- `index/` - Main application pages

## Setup (XAMPP)

1. Start Apache and MySQL in XAMPP.
2. Open phpMyAdmin.
3. Import `database/school_fee.sql`.
4. Put this project inside `htdocs` (or open using your local PHP server setup).
5. Open browser and go to:
   - `http://localhost/school_proj/index/`
